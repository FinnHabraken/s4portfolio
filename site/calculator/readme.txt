Deze versie van de calculator is een verouderde versie, ik heb hem ondertussen aangepast zodat hij werkt zonder Jquery en extra features heeft.
Calculator
De code wordt elke keer uitgevoerd wanneer er op een knop gedrukt wordt
 
Zodra je op een knop drukt wordt er eerst gekeken naar de tekst die er op die knop staat. Als het een nummer is dan wordt <p1> vervangen door dit nummer, tenzij hier al een nummer staat, want dan voegt hij alleen een nummer toe aan <p1>. Als de knop een + is dan wordt er een + ingevuld in <p2>. Als de input een ‘=’ is, dan wordt de code uitgevoerd
 
De onderste codevoert de uitrekening van num1 en num2 uit, zodat het uiteindelijke antwoord weergegeven wordt op <p4>.

De html hieronder zorgt ervoor dat de functie uitgevoerd wordt zodra er op een van de buttons geklikt wordt
